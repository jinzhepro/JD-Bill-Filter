# 京东万商对帐单处理系统 - Next.js 版本

一个基于 Next.js + React + Tailwind CSS 的现代化 Web 应用，用于自动处理京东万商对帐单 Excel 文件，根据业务规则自动过滤、统计和整理数据。

## 功能特性

- 📁 **文件上传**: 支持拖拽上传和点击选择文件（Excel/CSV）
- 🔍 **智能数据解析**: 自动解析 Excel/CSV 文件并验证数据结构
- 🚫 **自动过滤**: 自动删除费用项为"直营服务费"的数据行
- 📊 **订单统计**: 统计所有单据类型为"订单"的商品数量
- 🔄 **退款扣减**: 根据商品名称扣减"取消退款单"和"售后服务单"的数量
- 💰 **商品单价设置**: 为每个商品设置单价，支持默认单价
- 🧮 **总价自动计算**: 自动计算单价 × 最终数量的总价
- 📈 **处理统计**: 提供总数量和总金额的统计信息
- 👀 **数据预览**: 支持处理结果预览
- 💾 **简化导出**: 生成包含 5 个字段的 Excel 文件（商品名称、商品编号、单价、数量、总价）
- 🎨 **现代化界面**: 响应式设计，优雅的用户体验
- ⚡ **实时反馈**: 处理进度和日志实时显示

## 技术栈

- **框架**: Next.js 14+ (App Router)
- **UI 库**: Tailwind CSS
- **语言**: JavaScript
- **状态管理**: React Hooks + Context API
- **文件处理**: SheetJS (xlsx)
- **构建工具**: Next.js

## 使用方法

### 1. 上传文件

- 点击"选择文件"按钮或直接拖拽文件到上传区域
- 支持 `.xlsx`、`.xls` 和 `.csv` 格式
- 文件大小限制：50MB

### 2. 设置商品单价

- 文件上传成功后，系统自动处理数据并显示商品单价设置界面
- 系统显示所有最终数量大于 0 的商品
- 为每个商品设置对应单价
- 支持预设商品的默认单价
- 所有商品必须设置单价才能继续处理

### 3. 处理数据

- 所有商品设置完成后，系统自动开始处理
- 处理步骤包括：
  - 应用单价到商品数据
  - 计算每个商品的总价
  - 生成最终统计结果

### 4. 查看结果

- 处理完成后显示总数量和总金额统计
- 可以点击"预览数据"查看处理结果（前 100 行）
- 点击"下载处理后的 Excel"获取最终文件

## 处理逻辑

### 数据处理流程

1. **删除直营服务费**: 自动删除所有费用项为"直营服务费"的数据行
2. **订单统计**: 统计所有单据类型为"订单"的商品数量，按商品名称分组
3. **退款扣减**: 根据商品名称扣减"取消退款单"和"售后服务单"的数量
4. **单价应用**: 为每个商品设置单价并计算总价
5. **结果生成**: 生成简化的统计结果

### 计算规则

- **最终数量** = 订单数量 - 退款扣减数量
- **总价** = 单价 × 最终数量
- 只处理最终数量大于 0 的商品

## 数据结构要求

Excel/CSV 文件必须包含以下列：

| 列名     | 说明       | 示例                               |
| -------- | ---------- | ---------------------------------- |
| 订单编号 | 唯一标识符 | "202312280001"                     |
| 单据类型 | 订单状态   | "订单"、"取消退款单"、"售后服务单" |
| 费用项   | 费用分类   | "直营服务费"、"货款"               |
| 商品编号 | 商品标识   | "PROD001"                          |
| 商品名称 | 商品名称   | "iPhone 15"                        |
| 商品数量 | 商品数量   | 1, 2, -1                           |

**输出文件结构**

处理后的 Excel 文件包含以下 5 个字段：

| 列名     | 说明        | 示例        |
| -------- | ----------- | ----------- |
| 商品名称 | 商品名称    | "iPhone 15" |
| 商品编号 | 商品编号    | "PROD001"   |
| 单价     | 商品单价    | 10.50       |
| 数量     | 最终数量    | 5           |
| 总价     | 单价 × 数量 | 52.50       |
